A digital console based bank application having functionalities of  Account Management (Open, Close accounts)
• Deposit & Withdrawal Operations
• Fund Transfer between accounts
• Transaction History & Statement Generation
• Search & Filter functionality
• Custom Exception handling
• Data Validation best practices



It includes the Concepts like OOP Principles (Encapsulation, Abstraction)
  1.Java Records (Modern Java)
  2.Collections (ArrayList, HashMap)
  3.Exception Handling (try-catch, custom exceptions)
  4.Scanner for user input
  5.LocalDateTime for timestamps
  6.Repository Pattern for data management
  7.Service Layer for business logic
